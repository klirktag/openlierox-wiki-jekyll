###### Description

A sound resource is the data created for the waveform sound heard to reproduce the same effect when needed.

* * File types supported by Gusanos:** .wav, .ogg.

If there are 2 files with the same name but with different extentions, the one with the .ogg extention is used first.

Reserved files used by Gusanos (Shown without their extentions):

```
 * chat - Used when the message is sent to another player in-game.
```
