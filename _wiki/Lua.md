[Lua](http://www.lua.org) is a scripting language. We have started using it since we [merged Gusanos into OLX](/wiki/index.php/Merge_with_Gusanos/). It can be used now for any [map](/wiki/index.php/Gusanos_Maps/) or [mod](/wiki/index.php/Gusanos_Modding/). Check out the [list of all Lua classes / functions](/wiki/index.php/Lua_classes/).
