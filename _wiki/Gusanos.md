[Gusanos](http://gusanos.sf.net) is another [Liero](/wiki/index.php/Liero/) clone. Since OpenLieroX 0.59, we have [merged the Gusanos codebase into OpenLieroX](/wiki/index.php/Merge_with_Gusanos/).

[Read more about the game Gusanos here in the original Liero wiki](http://comser.liero.org.pl/wiki/index.php?title=Gusanos).

# Features

* [Lua](/wiki/index.php/Lua/) scripted mods / maps
* dynamic lights and shadows
* 3D sounds

# Documentation

Write your own mods/weapons: [Gusanos Modding](/wiki/index.php/Gusanos_Modding/)

Create your own maps: [Gusanos Maps](/wiki/index.php/Gusanos_Maps/)

Technical details about Gusanos in OpenLieroX: [Merge with Gusanos](/wiki/index.php/Merge_with_Gusanos/)
