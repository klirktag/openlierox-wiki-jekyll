[SourceForge](http://sourceforge.net) is a free service to host an Open Source project and provides all kind of tools to work on it, like [Git](/wiki/index.php/Git/), all the file release packages, the [mailinglists](/wiki/index.php/Mailinglist/) and so on.

[OpenLieroX](/wiki/index.php/OpenLieroX/) uses SourceForge since the very beginning for most of the [development](/wiki/index.php/Development/). As of May 2011 though, we have moved the [main code](/wiki/index.php/OpenLieroX_source_code/) [Git](/wiki/index.php/Git/) repository to [GitHub](/wiki/index.php/GitHub/).
