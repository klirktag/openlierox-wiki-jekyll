**The Basic User Page**

**How(to)**
In your user page (click your name under the box that has the header **Personal Tools** to your right.), click edit and paste this code in:

```` ```
{{UserPage|
|Name=Your Name
|Location=Your Location
|DOB=Date of Birth
|Date=Started Playing
|Set=Favorite Setting
|Nicks= Other nicks
|Clans= Clans
|Files=Files you created
|Awards=Your awards
|Other=Any other information about yourself you would like to add.
}}
``` ````

Edit all the information after the equal signs.

You can space and enter between each line, so its possible to have this:

```` ```
|Nicks=
*Nickname
*Nickname 2
*Nickname 3
|Awards=
*1st place here
*1st place there.
Other=
Lots of text
Lots and lots of text
Notice how you can keep going.
``` ````
