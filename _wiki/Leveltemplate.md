Use this template to create a basic layout for posting a level. Use the code fround below in a page you wanna make about a level and fill in the blanks.```` ```
{{LevelPage|
|LEVELNAME=Name of level
|IMAGENAME= Name of the image you uploaded. (Example:Lamda Bunker.jpg
|IMAGETEXT= Text that will display under image.
|INFO1=Text displayed under header and to left of image.
|LAYOUT=Text describing the layout of the level.
}}
``` ````
