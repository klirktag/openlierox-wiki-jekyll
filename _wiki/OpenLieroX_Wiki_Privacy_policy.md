This is an archival re-creation of the original OpenLieroX Wiki, a fan wiki for
the game [OpenLieroX](/wiki/index.php/OpenLieroX/). The original wiki, and the
account system it used, no longer exist.

This static re-creation does not run any server-side software, does not use
cookies, and does not collect any personal information from visitors. It is
hosted as static files on GitHub Pages; refer to GitHub's own privacy policy
for information about the hosting platform.

The article content was reconstructed from public snapshots stored in the
[Internet Archive Wayback Machine](https://web.archive.org/web/2011*/openlierox.net/wiki/).
