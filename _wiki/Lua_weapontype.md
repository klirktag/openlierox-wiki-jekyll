###### Class WeaponType

##### See also:

[weapon_random](/wiki/index.php/Lua_global/#weapon_random), [Weapon:type](/wiki/index.php/Lua_weapon/#type), [Player:select_weapons](/wiki/index.php/Lua_player/#select_weapons)

##### ammo

```
 ammo()
```

Returns the amount of [ammo](#ammo) for this weapon type when reloaded.

##### name

```
 name()
```

Returns the [name](#name) of this weapon type.

##### next

```
 next()
```

Returns the [next](#next) weapon type after this one.

##### prev

```
 prev()
```

Returns the previous weapon type after this one.

##### reload_time

```
 reload_time()
```

Returns the time it takes for this weapon type to reload.
