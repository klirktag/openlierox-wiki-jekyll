**GitHub** is a website for Open Source software development projects much centered around [Git](/wiki/index.php/Git/): [github.com](https://github.com)

As of May 2011, we have moved our [main code](/wiki/index.php/OpenLieroX_source_code/) repository to GitHub. Earlier, we have used [SourceForge](/wiki/index.php/SourceForge/) which is a similar site with much different priorities though. We also still use SourceForge for other services like file hosting.
