You can create your own theme/frontend for [OpenLieroX](/wiki/index.php/OpenLieroX/). OpenLieroX automatically searches the themes-directory for all themes and will list them in the combobox in the main menu for selection. A theme is basically just an overlay over the normal files. You can replace all menu graphics and other options this way. "- Default -" in the theme selection will not use any special theme overlay and thus fall back to the default theme.

See the provided frontends as examples/references.
