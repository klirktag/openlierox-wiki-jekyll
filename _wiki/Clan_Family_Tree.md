# The LieroX Clan Family Tree

The family tree is a project started by [AnubisBlessing](/wiki/index.php/User_AnubisBlessing/) which has as it's goal to map the development of clan creation in the Liero Xtreme community.

## Progress

Note:

* The mapping is not assumed to be complete, but rather shows what clans have been added to the map.
* Inheritance is indicated by the following syntax:

**Decendant** <- Ancestor1(Founder1) <- Ancestor2(Founder2) <- ... <- Ancestorn(Foundern)

* Multiple arrows indicate multiple inheritance, not a chain.
* If a clan has no known ancestors, it will be called a mother clan.
  * "No known ancestors" can indicate that research has not gotten that far, or simply that the clan is an genuine mother clan.
