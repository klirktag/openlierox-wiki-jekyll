GPL is short for General Public Licence. [OpenLieroX](/wiki/index.php/OpenLieroX/) uses a special type of this licence: the Lesser General Public Licence (LGPL). Actually, we never differed so much if we use GPL or LGPL and it also doesn't make such a big difference because there aren't really any other projects which depend on OpenLieroX.

* [GPL on Wikipedia](http://en.wikipedia.org/wiki/GNU_General_Public_License)
* <http://www.gnu.org/copyleft/lesser.html>
